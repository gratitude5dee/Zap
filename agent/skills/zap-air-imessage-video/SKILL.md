# Air iMessage Video

This is an internal Zap recipe used to keep Air's durable run records
resumable. It is private and must not be added to the public Zap registry.
Pricing is operator-configured from the current GMI console before work can
be planned or submitted. Its generated MP4 output requires either a connected
Vercel Blob store (`BLOB_STORE_ID` with managed OIDC) or the legacy
server-only `BLOB_READ_WRITE_TOKEN`.
