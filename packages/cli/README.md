# @wzrdtech/zap

Agent-first command line tools for Zap recipes.

```bash
npx @wzrdtech/zap@0.2.1 init my-zap-app
npx @wzrdtech/zap@0.2.1 new product-reveal
npx @wzrdtech/zap@0.2.1 validate agent/skills/zap-product-reveal/Zap.md
npx @wzrdtech/zap@0.2.1 run agent/skills/zap-product-reveal/Zap.md --json
```
