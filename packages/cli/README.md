# @wzrdtech/zap

Agent-first command line tools for Zap recipes.

```bash
npx @wzrdtech/zap@0.1.0 init my-zap-app
npx @wzrdtech/zap@0.1.0 new product-reveal
npx @wzrdtech/zap@0.1.0 validate agent/skills/zap-product-reveal/Zap.md
npx @wzrdtech/zap@0.1.0 run agent/skills/zap-product-reveal/Zap.md --json
```
